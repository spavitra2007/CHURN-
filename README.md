# CHURN-
The raw dataset was first cleaned and preprocessed, followed by feature engineering to create meaningful predictive variables. During the process, additional refinement was performed to eliminate potential data leakage and ensure model integrity.

Since the dataset was highly imbalanced (uneven churn vs. non-churn distribution), appropriate data cleaning and balancing considerations were applied to improve reliability.

After completing preprocessing and validation steps, the final cleaned CSV was generated and used for model training and further analysis

## 📂 Dataset Information

### 🔹 Raw Data
- File: `data/ChurnZero_dataset_v1.csv`
- Description: Original unprocessed churn dataset

### 🔹 pre-Processed Data
- File: `data/customer_full_backup.csv`
- Description: Cleaned dataset after preprocessing, feature engineering, and leakage removal
- ### Final data
- file-  data/filename.csv
- Description : final dataset with 38 features and model training ready dataset
